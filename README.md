
# xp-css-react

React components which use the HTML and CSS provided by https://botoxparty.github.io/XP.css/

## install

```
npm i xp-css-react
```

## examples

```
<WindowWithTitleBar text={"example"}>
  <p>Example</p>
</WindowWithTitleBar>
```
